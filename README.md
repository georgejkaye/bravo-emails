# bravo-emails

Sending automated emails for the University of Birmingham Theory Group PhD seminar, Bravo.

## Configuration

To set up the script, you must fill out a `config.json` with relevant data.

```json
{
  "page": "",
  "smtp": {
    "host": "",
    "port": "",
    "user": "",
    "password": "", 
  },
  "sender_email": "",
  "recipient_email": "", 
  "zoom": {
    "link": "",
    "id": "",
    "password": "",
  },
  "room": "",
}
```
